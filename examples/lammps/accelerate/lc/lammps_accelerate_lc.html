LAMMPS: Accelerate/in.lc¶

Reference: https://github.com/lammps/lammps/blob/master/examples/accelerate/in.lc
Required files: None
Description: Gay-Berne benchmark. Biaxial ellipsoid mesogens in isotropic phase. Shape: 2 1.5 1, cutoff 4.0 with skin 0.8, NPT, T=2.4, P=8.0

from __future__ import print_function
import random, math
from lammps import IPyLammps
L = IPyLammps()
